# تعليمات المشروع

## الهدف

بناء موقع عربي احترافي ومتجاوب لخدمات رقمية باستخدام الذكاء الاصطناعي.

## التقنيات

- HTML5
- CSS3
- JavaScript
- لا تستخدم أي framework دون طلب موافقة.
- اتجاه الموقع من اليمين إلى اليسار RTL.

## قواعد العمل

- استخدم HTML دلاليًا وواضحًا.
- اجعل التصميم متجاوبًا مع الهاتف والكمبيوتر.
- حافظ على سهولة قراءة الكود.
- لا تكرر CSS دون حاجة.
- استخدم أسماء واضحة للمتغيرات والدوال.
- افحص أخطاء JavaScript بعد كل تعديل.
- لا تضع كلمات مرور أو مفاتيح API في الملفات.
- لا تحذف أي قسم قائم دون توضيح السبب.
- اعرض ملخصًا للملفات التي عدلتها بعد كل مهمة.

## التحقق

بعد كل تعديل:

1. افحص بنية HTML.
2. افحص توافق التصميم مع الهاتف.
3. افحص Console بحثًا عن أخطاء.
4. راجع التغييرات باستخدام git diff.
